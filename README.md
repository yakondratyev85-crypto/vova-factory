# Vova Factory — генератор лабиринтов

Одностраничное веб-приложение для автоматической генерации лабиринтов для детских журналов.

## Возможности

- Несколько видов лабиринтов: прямоугольный, круговой, треугольный, гексагональный, медовые соты, островной, картинный и маршруты между домиками.
- Уровни сложности: лёгкая, средняя, сложная и экспертная.
- Тематическое оформление: космос, джунгли, море, сладости, динозавры и сказка.
- Вариации генерации: классический один путь, петли, длинные коридоры и комнаты.
- Декоративная иллюстрация вокруг каркаса лабиринта и поворот макета на 45°, -45° или 90°.
- Отображение решения для всех форм и отдельный режим соединения от 1 до 6 домиков цветными маршрутами.
- Экспорт готового макета в PNG.

## Проверка

```bash
npm test
```

Команда проверяет синтаксис `app.js` и smoke-рендер всех видов лабиринтов с включённым решением.

## Запуск

Откройте `index.html` в браузере или запустите локальный сервер:

```bash
python3 -m http.server 4173
```

Затем перейдите на `http://127.0.0.1:4173/index.html`.
