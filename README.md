# Vova Factory — генератор лабиринтов

Одностраничное веб-приложение для автоматической генерации лабиринтов для детских журналов.

## Возможности

- Несколько видов лабиринтов: прямоугольный, круговой, треугольный, гексагональный, островной и картинный.
- Уровни сложности: лёгкая, средняя, сложная и экспертная.
- Тематическое оформление: космос, джунгли, море, сладости, динозавры и сказка.
- Декоративная иллюстрация вокруг каркаса лабиринта.
- Отображение решения для всех видов лабиринтов и экспорт готового макета в PNG.

## Запуск

Откройте `index.html` напрямую в браузере — сайт не требует сборки и внешних зависимостей. Для проверки через локальный сервер выполните:

```bash
python3 -m http.server 4173
```

Затем перейдите на `http://127.0.0.1:4173/index.html`.
