# Maze Worksheet Studio

Самая простая локальная версия генератора детских лабиринтов.

В проекте теперь только один рабочий файл:

```txt
index.html
```

## Как запустить

1. Откройте папку проекта.
2. Дважды кликните по `index.html`.
3. Файл откроется в браузере.
4. Нажмите `Generate Maze`.

Ничего устанавливать не нужно. Терминал не нужен. Сервер не нужен. Сборка не нужна.

## Что есть на странице

- `Maze Worksheet Studio`.
- Сложность 8 × 8, 12 × 12 или 18 × 18.
- Seed.
- Rotation Angle.
- Theme: basic, space, forest, cats.
- Visual Mode: color, coloring, minimal.
- Generate Maze.
- Show Solution.
- Hide Solution.
- Download SVG.
- Download PNG.

## Как проверить через GitHub Desktop

1. Откройте репозиторий в GitHub Desktop.
2. Нажмите `Show in Finder` или `Show in Explorer`.
3. Откройте `index.html` в браузере.
4. Проверьте генерацию, seed, решение, поворот и скачивание.
5. Сделайте commit.

## Важно

В проекте нет установочных файлов и папок с зависимостями. Для проверки нужен только браузер.
