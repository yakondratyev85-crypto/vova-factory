# Vova Factory — генератор чистых лабиринтов

Одностраничное веб-приложение для автоматической генерации чистых печатных лабиринтов для детских журналов, заданий и постеров.

## Возможности

- Чистый визуал без картинок, эмодзи и персонажей: только геометрия, линии, вход, выход и опциональное решение.
- Формы: прямоугольный, идеально круговой, треугольный, соты/гексагональный с шестью математическими сторонами, прямолинейная спираль, кристалл с прямыми гранями, плетёный и ломаный ортогональный лабиринты.
- 10+ функций: сложность, seed, палитры, поворот готового лабиринта, толщина линий, открытые вход/выход, инверсия, направляющая сетка, показ решения, PNG/SVG-экспорт и генерация AI-промпта.
- Круговой режим строится в квадратном поле с одинаковыми радиусами, поэтому круг остаётся кругом, а не овалом.
- AI-блок не отправляет данные во внешние сервисы и не требует ключей: он готовит промпт, который можно скопировать в GPT/image generator для красивой последующей визуализации.

## Запуск

Откройте `index.html` напрямую в браузере — сайт не требует сборки и внешних зависимостей. Для проверки через локальный сервер выполните:

```bash
python3 -m http.server 4173
```

Затем перейдите на `http://127.0.0.1:4173/index.html`.
